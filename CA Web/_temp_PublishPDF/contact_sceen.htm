Contact Sceen

The Contact Screen contains a Search tab, an Add tab and an Advanced Search tab. When looking up information the Search or Advanced Search tabs will be used. When establishing a new Contact, the Add tab will be used.


Color Coding for the Contacts is selected by each property. In this example the colors selected represent the following:
Blue – Preferred Vendors
Red – Closed or do not use contacts
Black – Phonebook usage
