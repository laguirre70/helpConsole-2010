Internet Explorer Configuration

In this example we will use the site “Demo” as the site to be configured. The URL for the site “Demo” is https://demo.gksca.com

Site https://demo.gksca.com should be included in
 
• Internet Options>Security>Trusted Sites

 

• Tools>Pop-up Blocker>Pop-up Blocker Settings



Optional Internet Security Settings

When login in the site https://demo.gksca.com I get the pop-up message below:



I can simply click “Yes” (twice) and I will not see the message again until the site is reloaded or I login again. If I decide that I don’t want to see the message at all. I can configure the way “mixed content” is displayed by adjusting my internet security settings at Internet Options>Security>Internet>Custom Level>Miscellaneous>Display mixed content>Enable:




Printing Plug-in (Adobe Reader)

In order to be able to print any collateral it is necessary to have Adobe Reader installed in the client. If Adobe Reader is not installed in the client CA Web will provide you with a link to download the latest version of adobe reader when trying to print.
