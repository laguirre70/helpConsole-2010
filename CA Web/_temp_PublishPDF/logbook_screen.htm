Logbook Screen




The Logbook display screen contains a Search tab, an Add tab and an Advanced Search tab.  When looking up information the Search or Advanced Search tab will be used.  When establishing a new Contact, the Add tab will be used.
 
The Logbook entries may be color coded to indicate their status of completion. Each property may determine the color selections for each status. The following are the color designations in this example:
Blue – Pending Logbook Entry
Red – Not completed Guest Request
Black – Completed Guest Request
Purple – On-hold Guest Request
