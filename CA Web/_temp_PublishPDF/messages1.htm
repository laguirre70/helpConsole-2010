Messages Overview

 
 
 
 

Messages are an inter-staff communication.

From the “Message Tab” you can type in any message and send it to either staff member or a group (or department)of staff members.

First select Create Message > Select the staff members or Message Group that you would like to send it to and then click Send.



 
 
 
 

To Read you messages:

Go to Messages> click on the Message> click View

 

This will then bring up the message:


You have many options that you can do from here:

1.      Reply  back to the person that sent it

2.      Reply All to everyone copied on the message

3.      Froward it on to someone else

4.      Print it

5.      Close it.
