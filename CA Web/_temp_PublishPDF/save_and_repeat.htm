Save and Repeat

When making repeat reservations for the same guests, using the Save and Repeat shortcut eliminates the extra steps that would be required if each entry were saved and closed.




When the Save and Repeat button is clicked, a new logbook entry is created with the exact same guest information that was contained in the old logbook entry as seen below.




Now the user can finish the logbook entry with the next request and so on until all the requests have been entered.
