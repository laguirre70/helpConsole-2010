Firefox Printing Issue

Issue:
CA Web not able to open the print dialog box when attempting to print any report using Mozilla Firefox.

Solution:

The issue appears to be caused by the default settings in Firefox to use the native Firefox viewer for PDFs. here is the recommendation suggested to fix the issue in FF:  

  1. Open Firefox
  2. Change setting: [Alt] -> Tools -> Options -> Applications > Portable Document Format (PDF) -> Use Adobe Acrobat option
