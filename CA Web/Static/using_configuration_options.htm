Using Configuration Options

To manage the configuration options go to Administration>Options>Configuration Options.



The Configuration Options are:

  1. Site Preferences - This information is used for the page name, time zone, some report headings and the Google maps API address (where available).
  2. Profiles settings - To select which columns will be displayed in the profiles screen grid view.
  3. Logbook Settings - Today button options and Contact fields link options.
  4. Language Settings - Default language.
  5. Form and Report Font settings for Contacts, Logbook and Profiles.
  6. Guest Web Interface - Registration Fields.
  7. Residential Settings.
  8. Guest Web Interface - Email from settings.
  9. Contacts displayed settings
