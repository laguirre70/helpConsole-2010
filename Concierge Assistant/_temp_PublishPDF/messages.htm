Messages

Improve communications between shifts, concierge, and departments. A personal diary for each staff member to save important messages until no longer needed. Trace file for all logbook, event and contact entries to ensure accountability. Ability to “Pass On” and “Forward” messages

The Concierge Assistant Messaging system is built on the ability to send three distinct types of messages:

  1. Plain Text
  2. Logbook Messages
  3. Event Notifications
  4. Contact Notifications
  5. Profiles Notifications
