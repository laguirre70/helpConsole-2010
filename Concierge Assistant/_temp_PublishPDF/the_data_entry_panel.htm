The Data Entry Panel


The Data Entry Panel is a powerful feature where you can define exactly what information you want the Concierge Staff to obtain for each type of logbook entry. You have a maximum of 15 labels to define for each logbook category.

To create a new Data Entry Panel simply click on the position of the label key the label text into the label field, and press Enter or click on Add. The cursor will automatically skip to the next label line. Repeat the process until all labels are defined. 




To delete a Data Entry Panel label you select the label to be deleted and press the Delete button.

If you know that another Data Entry Panel has been created that is like what you wish to have for this logbook category we have the "Clone" feature.


Simply select the Data Entry Panel from the Drop Down List and Press the Clone Button. The selected Data Entry Panel will appear. You may add or change any label lines you wish.
