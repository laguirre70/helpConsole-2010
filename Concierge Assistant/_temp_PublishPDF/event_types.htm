Event Types

The Event Types are used to categorize your Event entries.    For example, you will want to be able to look up all "Conventions" occurring this week. To do this you will want to have the category "Conventions" available as an Event Type selection.



To add a new Event Type simply key the type into the Event Type field.    Press Enter or click on Add Type and you have a new Event Type.

If you do not like a particular Event Type Name you may change it by selecting the original Event Type and keying the new description of the event type into the "Replace With" field.    When you change this field the Add button changes to a "Replace" button.    Press the "Replace" button and all Event entries using this code will be updated with the new event type.

To delete an Event Type you select the type to be deleted and press the Delete button.    If no Events are coded with this Event Type the type will be deleted.
